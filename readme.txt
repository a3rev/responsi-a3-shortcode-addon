Install this plugin to add extended Shortcodes on the Responsi Framework.

== Description ==
Responsi Shortcodes Add-on extends the existing Responsi Framework Shortcodes and is an essential tool for creating stunning content without writing code 

= Shortcodes =
* Fontface Icons - 360+ icons that can be inserted and styled by Shortcode.
* Flip Boxes - Insert and style Flip Boxes by Shortcode.
* More coming soon. 

== Installation ==
Minimum Requirements
* WordPress 4.1
* PHP version 5.2.4 or greater
* MySQL version 5.1 or greater


== Usage ==
1. Install the Responsi Framework
2. Install and activate a Responsi Child Theme.
3. Install and activate the Responsi Shortcode plugin from the Responsi menu.
4. Open the Shortcode menu from the Responsi main menu
5. Select the Shortcode type from the Shortcode Main menu.
6. Set default settings. Can be edited from the insert Shortcode pop up menu on any page or post.
